# Code of Conduct

## Our Commitment

We are committed to providing a welcoming, respectful, and inclusive environment for everyone.

## Expected Behavior

- Be respectful.
- Be constructive.
- Welcome different ideas.
- Help others learn.

## Unacceptable Behavior

- Harassment
- Discrimination
- Personal attacks
- Offensive language

By participating in this project, you agree to follow this Code of Conduct.
